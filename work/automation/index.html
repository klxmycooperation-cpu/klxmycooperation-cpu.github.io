Как собирается / 04

Не конвейер.
Последовательность решений.

  1. 01

    Разбираю задачу

    Изучаю ручные действия, существующие программы, документы и желаемый результат.

  2. 02

    Проектирую решение

    Определяю, откуда приходят данные, как они проверяются и между какими системами передаются.

  3. 03

    Собираю рабочую версию

    Реализую основной обмен и интерфейс, в котором сотрудник видит ход операции.

  4. 04

    Проверяю и дорабатываю

    Тестирую ошибки, повторные операции, целостность данных и поведение интеграций.

  5. 05

    Запускаю и развиваю

    Подготавливаю управляемый процесс к использованию и дальнейшему подключению новых сценариев.

  • REST API
  • Microsoft SQL Server
  • PowerShell
  • Delphi
  • ЭЦП и CryptoPro
  • XML и ЭДО
